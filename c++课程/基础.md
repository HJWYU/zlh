# 语句

- ## 分支		

  1. if

     ```c++
     if(条件){
     	主体；
     }			
     ```

  2. if else

     ```c++
     if（条件1）{
     	主体1；
     }else {
     	主体2；
     }	
     ```

  3. else if

     ```c++
     if(条件1){
     	主体；
     }else if(条件2){
     	主体1；
     }else if (条件3){
     	主体2；
     }else {
     	主体3；
     }
     ```

     

  4. swith

     ```c++
     swich(选择句){
     	case 标签1：//struct
     				break；
         case 标签2：//struct
     				break；
         default：//struct
     }
     ```

     

- ## 循环		

  1. for	//固定增加步骤，如枚举

     ```c++
     for (初始化；判断条件；更新) {
         //struct
         
     }
     ```

     

  2. while //只确定循环终止状态

     ```c++
     while (判断条件){
         //struct
         
     }
     ```

     

  3. do.....while //先执行再判断

     ```c++
     do {
         //struct
     }while(判断条件)；
     ```

  4. break/continue

     break //退出循环

     continue //退出余下部分

- ## 指针 //存储地址的变量

  对应类型所占大小的空间的起始地址

  声明与使用

  ```c++
  int main(){
      int a = 123;//a=123
      int* pa = &a;
      *pa = 321;//a=321
  }
  //空指针
  nullptr
      
  int* p = new int(1234);
  delete p;
  
  //arr
  int lenth = 2;
  int* p = new int[lenth];
  delete[] p;
  
  
  //二维数组
  int** a = new int*[5];
  for (int i = 0; i < 5; i++) {
      a[i] = new int[5];
  }
  for (int i = 0; i < 5; i++){
      delete[] a[i];
  }
  delete[] a;
  ```
